# Urban Infrastructure Failure Prediction

This project develops machine learning models to predict infrastructure failure in urban environments.

## Objective
To classify whether a city block will experience infrastructure failure using environmental and structural features.

## Methodology
- Data cleaning and preprocessing
- Train/Validation/Test split (60/20/20)
- Feature scaling and encoding
- Models used:
  - Logistic Regression
  - Decision Tree
  - Neural Network
- Evaluation metric: ROC-AUC (due to moderate class imbalance)

## Key Insight
Infrastructure age and environmental stress (e.g. moisture, debris) are strong predictors of failure risk.
